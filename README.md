# Watery Wails (working title)

A simple Five Nights At Freddy's / I'm On Observation Duty style game written in C++ for the purpose of both exploring a different style of story telling as well as expanding on my skills with C++.

The #1 objective of this project is for me to strengthen the skills I have and build new ones that I didn't learn from my classes in school. That being said, if you want to check out this repo, fork, and or submit a pull request, I am happy for that. So long as there is the understanding that this is a learning adventure for me that I hope will result in a game that I can distribute and sell.

Additionally, the actual story of the game will involve very dark themes. While much of the code will be unrelated, I want to put a disclaimer here just in case.
